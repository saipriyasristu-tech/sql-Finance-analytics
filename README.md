# sql-Finance-analytics Project

## Project Overview
This project analyzes financial sales data using SQL queries and Excel reporting to generate business insights.

## Tools Used
- SQL
- Excel
- GitHub

## Analysis Performed
- Gross sales report
- Monthly product transactions
- Total sales amount analysis

## Dataset
The dataset used in this project was provided as part of the Codebasics Data Analytics Bootcamp and is not included in this repository.

## Files Included
- SQL queries
- Excel financial report
- Screenshots of results
